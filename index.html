Top Music Charts Web Application

The Top Music Charts web application utilizes python and the web framework Django to build a web application that ranks the top ten songs of the decade from the past 70 years. Through API's and PostgreSQL database management, users are able to be created, discussion posts can be made as well as comments on those posts. This site was also deployed to AWS for a short time.
